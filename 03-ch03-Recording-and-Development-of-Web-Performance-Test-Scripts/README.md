#### 1.执行顺序
配置元件-前置处理器-定时器-采样器-后置处理器-断言-监听器
#### 2.作业域
* 分层规则：监听器、配置元件、前置后置处理器、断言、定时器
* 顺序规则：逻辑控制器、采样器
#### 3.Jmeter属性和变量（大小写敏感）
* 属性：在配置文件jmeter.properties中
* 变量：线程不安全
